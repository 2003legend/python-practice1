# 我的 Python 学习笔记

## Git
Git 是本地版本管理工具，用来记录代码修改历史。
GitHub 是远程托管网站，可以把仓库 push 上去备份。
常用命令：git add、git commit、git push。

## Python 基础
列表 list 类似 C++ 的 vector，用 append 添加元素。
字典 dict 类似 map，用 mp.get(key, 0) 避免 KeyError。
f-string 格式化：print(f"我是 {name}")。

## SQL
SELECT 查询，WHERE 筛选，JOIN 联表，GROUP BY 分组，HAVING 筛组。
LEFT JOIN 可以找出左表有、右表没有匹配的行。

## FastAPI
FastAPI 用来写后端 API，uvicorn 负责启动服务。
GET 用于查询，POST 用于创建，PUT 修改，DELETE 删除。
Swagger 文档在 /docs 路径。

## 大模型
大模型 API 是发送 prompt 到云端，返回生成文本。
RAG 是检索相关资料，拼进 prompt，再让模型回答，减少胡编。
